# 🔗 Blockchain Demo Application

## 📌 Overview

This project is a full-stack blockchain-based application developed to demonstrate the core principles of blockchain technology and cryptographic security. The system allows users to create transactions, generate blocks, and validate the integrity of the blockchain through a web-based interface.

## 🚀 Features

* 🔗 Blockchain structure with linked blocks
* 🔐 Cryptographic hashing using SHA-256
* ⛏️ Proof-of-Work (mining with nonce and difficulty)
* 💸 Transaction-based block creation
* ✅ Blockchain validation and tampering detection
* 🌐 Interactive frontend built with React (Vite + TypeScript)
* ⚙️ Backend API developed using Python Flask

## 🧠 Technologies Used

* Frontend: React, TypeScript, Vite
* Backend: Python, Flask
* Cryptography: SHA-256 hashing
* Deployment: Vercel (Frontend), Render (Backend)

## 🎯 Purpose

The main objective of this project is to provide a practical implementation of blockchain concepts such as hashing, block linking, and consensus mechanisms. It also demonstrates how cryptographic techniques ensure data integrity and security in distributed systems.

## 🔍 How It Works

1. Users input transactions through the web interface
2. Transactions are added into blocks
3. Blocks are mined using a Proof-of-Work algorithm
4. Each block stores the hash of the previous block
5. The system validates the blockchain to detect tampering

## 🛡️ Security Demonstration

The system includes a tampering simulation feature where modifying block data results in validation failure, illustrating the immutability property of blockchain.
